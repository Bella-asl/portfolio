# portfolio
Примеры учебных проектов и работ со стажировки

Дашборд со стажировки в ООО "Вин Солюшенс"
Платформа: Fastboard  
Источник данных: Excel-выгрузка
Задача: разработать дашборд по анализу посещаемости проектов на платформе Fastboard. Определить какие дашборды и потоки реально используются, сколько уникальных пользователей, какие проекты являются неподключенными, какие дашборды «мертвые», а какие — ключевые.

Для опубликования в проекте представлены сгенерированные данные. 
Необходимо было определить метрики, разработать KPI, фильтры, графики и диаграммы для анализа и визуализации.
В результате мной разработаны метрики и визуализации: количество активных пользователей в месяц (MAU), процент вовлеченности пользователей, статус и обновляемость проекта, динамика количества проектов и активных пользователей, востребованность конкретного дашборда. Построила систему фильтров (по дате, потоку, проекту, пользователю), создала дашборд с ключевыми графиками. На основе дашборда можно было принять решение об отключении неиспользуемых отчетов и оптимизации нагрузки на платформу.
Доступ к дашборду возможен при демонстрации экрана.
